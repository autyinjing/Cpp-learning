知识点总结：

1. 运算符重载：operator OP(argument-list); OP为运算符;
	优点：可以使同一个类的两个对象或一个类对象与一个内置类型直接进行运算;
	缺点：若要让一个类对象与一个内置类型变量运算，则对它们的位置有严格要求;

2. 友元函数：friend className operator OP(argument-list); (函数原型)
	优点：可以使内置类型变量与类对象运算时不用考虑参数位置，其设计可以使用返回语句来翻转操作数，最终变成非友元;
	缺点：

	注意：友元函数不是类方法，只是拥有类私有数据的访问权限，定义时不用加类修饰符，也不用加关键字friend;

3. 重载<<运算符：ostream & operator<<(ostream & os, const c_name & obj)
		 {
			 os << ...;
			 return os;
		 }
	使用返回语句的目的是使重载后的<<运算符可以处理一条语句上使用多个<<的情况;

4. 获得随机数的方法：srand(time(0));该语句在程序每次运行时都设置不同的种子;
		     rand();该语句用一个初始种子值来获得随机数;

5. 转换函数：operator typeName();
